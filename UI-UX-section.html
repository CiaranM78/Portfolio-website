Virtual and Mixed Reality portfolio

UI/UX Design for Virtual Reality Tellus Project (Academic)

Key Contributions

Interaction Design & Animation

Designed a more immersive hand extension mechanic to solve mismatches between real and virtual hand positions. Improved gesture fluidity using Unreal Engine’s input tracking for finger-hover detection. Developed mid-animation transitions and reverse cycling states for more natural hand motion. Proposed blendspace-based animation system for future refinement.

Ergonomic Testing & Spatial Layout

Conducted comparative user testing across different physical environments. Identified ideal spatial requirements (~1–2m range) for seated VR users to confidently interact with virtual objects. Designed test setups to analyze movement freedom and cognitive comfort during gameplay.

Usability & Cognitive Modeling

Focused on reaction times, animation realism, and player orientation to maintain immersion. Iterated designs based on peripheral vision insights, ensuring animations were smooth in “corner view” areas.
